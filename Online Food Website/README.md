## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set your Firebase values in `.env.local`.
   Optional: add `VITE_PARTNER_ADMIN_EMAILS=email1@example.com,email2@example.com` to grant admin panel access to partner accounts.
3. Run the app:
   `npm run dev`
